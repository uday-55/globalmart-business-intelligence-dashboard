# GlobalMart Business Intelligence Dashboard

## Project Overview
End-to-end Business Intelligence project using PostgreSQL, SQL and Power BI.

## Technologies
- PostgreSQL
- SQL
- Power BI

## Features
- KPI Dashboard
- Monthly Sales Trend
- Sales by Category
- Profit by Region
- Interactive Filters

## Key Insights
- Technology generated highest sales.
- West region generated highest profit.
- Profit margin ~12.47%.
