# 📌 Project Overview

Retail organizations generate thousands of transactions every day. However, raw transactional data alone does not provide meaningful business insights.

This project demonstrates a complete **Business Intelligence (BI) workflow** where retail sales data is:

* Collected and stored in PostgreSQL
* Analyzed using SQL
* Visualized through Power BI dashboards
* Converted into actionable business insights

The dashboard enables stakeholders to monitor:

* Revenue performance
* Profitability
* Regional trends
* Product category performance
* Customer behavior
* Business KPIs

---

# 🎯 Business Problem

Management wants answers to critical business questions:

### Revenue Analysis

* Which product categories generate the highest sales?

* Which categories are most profitable?

### Operational Analysis

* Are discounts impacting profitability?
* Which business segments perform best?

* How can decision-makers monitor business performance quickly?

# 🔍 SQL Analysis Performed

The project includes business-oriented SQL analysis such as:

### Category-wise Sales Analysis

```sql
SELECT
    category,
    SUM(sales) AS total_sales
FROM superstore_orders
GROUP BY category
ORDER BY total_sales DESC;
```

### Category-wise Profit Analysis

```sql
SELECT
    category,
    SUM(profit) AS total_profit
FROM superstore_orders
GROUP BY category
ORDER BY total_profit DESC;
```

### Regional Profitability Analysis

```sql
SELECT
    region,
    SUM(profit) AS total_profit
FROM superstore_orders
GROUP BY region
ORDER BY total_profit DESC;
```
